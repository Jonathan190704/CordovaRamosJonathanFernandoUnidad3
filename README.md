# PortalCore

## Ver la página en localhost

### 1. Instalar XAMPP

Descarga e instala [XAMPP](https://www.apachefriends.org/).

### 2. Copiar el proyecto

Copia esta carpeta dentro de:

```
C:\xampp\htdocs\
```

### 3. Iniciar Apache

Abre el **XAMPP Control Panel** y pulsa **Start** en Apache.

### 4. Abrir en el navegador

```
http://localhost/CordovaRamosJonathanFernandoUnidad3/index.html
```

Listo. La página principal ya debería verse.

> Si renombraste la carpeta, cambia la URL por el nombre que le pusiste.
